# 놀면서 운동하는 가면 놀이터
- 게임으로 운동해, 개운해는 운동에 진입장벽을 느끼는 현대인들에게 재밌고 뿌듯한 액티비티를 제공합니다 !



## 💡 주요 특징

- WebRTC 기반 비디오 공유 서비스
- WebSocket을 이용한 실시간 게임 정보 통신, 채팅
- OpenCV를 이용한 얼굴 이모지 처리
- Teachable machine을 이용한 자세 판별
- 활동기록 조회 및 보상 시스템
